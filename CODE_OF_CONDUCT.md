# Code of Conduct

## Our standard

NeuroCAD public repositories are maintained for serious technical discussion, integration clarity, and respectful collaboration.

We expect contributors and reviewers to:

- communicate clearly and professionally
- stay technical, factual, and respectful
- avoid harassment, abuse, or personal attacks
- keep public issues focused on the repository scope

## Scope

This code of conduct applies to:

- GitHub issues
- pull requests
- repository discussions and review threads

## Reporting

If you experience or witness unacceptable behavior in a NeuroCAD public repository, contact:

- `office@neurocad.eu`

## Enforcement

Repository maintainers may edit, lock, hide, or remove content that falls outside these expectations.
